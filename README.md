## opensajux
Personal Website Management software hosted on Google AppEngine. Developed using JSF, CDI and other Java technologies

## Technologies
+ Google AppEngine 1.7.4
+ Java Server Faces 2.0
+ PrimeFaces 3.4.2
+ Weld
+ Java Data Object
+ HTML / JavaScript
